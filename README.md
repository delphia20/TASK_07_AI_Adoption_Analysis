# Student AI Adoption and Career Readiness Analysis

## Objective

Analyze AI tool adoption, career readiness, learning behavior, and student trends.

## Dataset

Student AI Usage Dataset (15000 Records)

## Tools Used

* Python
* Jupyter Notebook
* Pandas
* Seaborn
* Matplotlib
* Scipy

## Project Phases

1. Data Understanding
2. Statistical Analysis
3. Trend Investigation
4. Correlation Analysis
5. Hypothesis Testing
6. Student Segmentation
7. Business Insights

## Conclusion

The analysis shows that AI tool adoption positively influences student learning, motivation, and career readiness. Students with higher AI usage tend to demonstrate better preparedness for future career opportunities.
